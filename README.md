## Memory game

**Description**
A memory card game where you need to pick unique pokemon each turn. But theres a catch: the pokemon will randomly shuffle after each turn.

**Features**

- API integration with the https://pokeapi.co/api/v2 to randomly choose 15 new pokemon each user
- Score tracking and high score count

**Technology**

- React
- Vite

**How to run**

`npm install`
`npm run dev` for dev | `npm run build` for prod
